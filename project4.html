Walmart Search Engine

Walmart gives out to the public only a limited number of API keys if you want to use an API to crawl their database system. Because of this, we built an entire web scraper using Selenium in Python. Of course the whole process is time consuming (about 3 hours for 5000 products with information details crawled), the data obtained was neat.

This project is still in my progress. We would use Natural Language Processing to process the text data and reduce the dimensions of the variables, embedding the words, train the model, create a GUI and host it on a website. The end result hopefully would be something like this: (image source)
